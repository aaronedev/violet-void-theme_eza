# Violet Void Theme for eza

A dark purple-tinted theme for [eza](https://github.com/eza-community/eza) (modern ls replacement).

## Install

```bash
mkdir -p ~/.config/eza
cp violet_void_theme.yml ~/.config/eza/theme.yml
```

Or symlink:
```bash
ln -s ~/path/to/violet-void-theme_eza/violet_void_theme.yml ~/.config/eza/theme.yml
```
